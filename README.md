# Herbarium-data-wrangling
R workflow for data restructuring and plant species data extraction habitat notes, standardizing species lists, updating datasets with georeferenced locations and standardising species name. 

# Plant species extraction & georeferencing from Mauritius Herbarium collection

## Overview
This project provides an R-based workflow to:
1. Clean data of duplicates and unreliabel informations
2. Extract plant species names from unstructured text
3. Fill missing coordinates from Gazetteer
4. Update/validate species names via U.Taxonstand
5. Create a density map of all collection points

The workflow is designed for extraction and cleaning of herbarium datasets.
